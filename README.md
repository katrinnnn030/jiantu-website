## 🎯 項目簡介

**健途** 是一個創新的AI驅動運動防護平台，透過人工智能技術為用戶提供個人化的運動傷害預防方案。我們結合物理治療專業知識與先進的AI動作分析，讓每個人都能安全、有效地享受運動樂趣。

## ✨ 主要功能

### 🎯 AI動作分析
- 實時動作捕捉與分析
- 運動姿勢糾正指導
- 潛在風險識別

### 📊 個人化評估  
- 基於用戶身體狀況定制方案
- 運動能力綜合評估
- 個人化訓練計劃生成

### 🛡️ 傷害預防
- 運動風險預警系統
- 預防性訓練指導
- 康復進度追蹤
